# Premium UI Principles

Premium UI is not a visual effect. It is the compound result of hierarchy, spacing, typography, content, color, media, interaction, and restraint.

## Principles

- Make the subject clear immediately.
- Use whitespace as structure, not emptiness.
- Use typography as the main design material.
- Use color strategically.
- Use cards only when they are meaningful containers.
- Avoid nested cards.
- Avoid generic gradients and blobs.
- Avoid one-note palettes.
- Avoid low-contrast elegance.
- Avoid over-rounded everything.
- Keep copy specific.
- Show real product, real work, or real content whenever possible.

## Inspiration Handling

Learn from Apple, Stripe, Linear, Vercel, Raycast, Framer, Supabase, Aceternity UI, Magic UI, Origin UI, Clerk, Notion, Figma, and Awwwards at the level of principles. Do not copy proprietary artifacts.
