# Ecommerce Example

Prompt:

```text
Create an ecommerce product page with clear product imagery, calm variant selection, add-to-cart feedback, accessible forms, and no scroll hijacking.
```

Acceptance criteria:

- Product remains the hero.
- Price, stock, variants, and shipping are clear.
- Add-to-cart feedback is immediate.
- Hover media does not obscure buying.
- Checkout path is calm.
